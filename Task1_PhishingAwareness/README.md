# Phishing Awareness Training

This project was created for the CodeAlpha Cyber Security Internship.

## Objective
To educate users about phishing attacks, fake emails, fake websites, and online safety practices.

## Topics Covered
- What is phishing?
- Types of phishing attacks
- Fake emails and websites
- Social engineering
- How to stay safe online
